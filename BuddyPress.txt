XAMPP環境下でのBuddyPress導入メモ

WordPressのインストール
	好きなディレクトリにWordPressをコピー

	PHPMｙAdminからデータベースを作成

	wp-config.phpの編集
		/** WordPress のためのデータベース名 */
		define('DB_NAME', 'dbname');

		/** MySQL データベースのユーザー名 */
		define('DB_USER', 'root');

		/** MySQL データベースのパスワード */
		define('DB_PASSWORD', 'Password');

		/** MySQL のホスト名 */
		define('DB_HOST', 'localhost');

		/** データベースのテーブルを作成する際のデータベースのキャラクターセット */
		define('DB_CHARSET', 'utf8');

	あとは指示に従ってポチポチ


BuddyPressのインストール
http://www.webcreatorbox.com/tech/wordpress-plugin-buddypress/
http://machiiku.jp/archives/585

	プラグインのダウンロード
		プラグイン > 新規追加から"BuddyPress"を検索
		インストールする

	日本語化
		http://sns.webmarketingsupporter.com/%E3%83%97%E3%83%A9%E3%82%B0%E3%82%A4%E3%83%B3%E7%BF%BB%E8%A8%B3/bpjpmo
		buddypress-ja.moをダウンロード
		/wp-content/plugins/buddypress/bp-languages/にアップロード

	ユーザーアカウント作成の設定
		設定 > 一般設定のメンバーシップ欄から「誰でもユーザー登録ができるようにする」を有効化

	wp-config.phpの編集
		define('WP_ALLOW_MULTISITE', true);を追加
		/* 編集が必要なのはここまでです ! WordPress でブログをお楽しみください。 */ より上に書く必要あり

	ネットワークを作成
		ツール > ネットワークの設置からネットワークを作成
		支持に従ってwp-config.php、.htaccessにコードを追加

	ユーザーアカウント新規登録の許可
		参加サイト > サイトネットワーク管理者 > 設定から新規登録許可を変更


登録確認メールを送る
http://ameblo.jp/nero-333/entry-11400757241.html

	php.iniを編集
		sendmail_path = "\"C:\xampp\sendmail\sendmail.exe\" -t"
		他の"sendmail_path"はコメントアウト

	sendmail.iniを編集
		[sendmail]
		error_logfile=error.log
		pop3_server=
		pop3_username=
		pop3_password=
		force_recipient=
		hostname=[自分のgmailアカウント]@gmail.com
		debug_logfile=debug.log

		; gmail
		smtp_server = smtp.gmail.com
		smtp_port = 587
		smtp_ssl = auto
		error_logfile = error.log
		auth_username = [自分のgmailアカウント]
		auth_password = [自分のgmailアカウントパスワード]
